## Touch Pads (Performance/Drum Pads)

This section is for using the Loupedeck’s touch buttons as performance/drum pads. In FL Studio’s **Performance Mode**, MIDI notes trigger clips on Playlist tracks. The mapping below follows FL’s default Performance Mode note layout ([Playlist in Performance Mode](https://www.image-line.com/fl-studio-learning/fl-studio-online-manual/html/playlist_performance.htm#:~:text=Note%20Number%20%2F%20Note%20Name,7%2C%20Clip%201%20to%2012)). You can also use these pads to play a channel (like a FPC drum kit or sampler) by sending MIDI notes.

**Note:** To use these, enable Performance Mode (Ctrl+P) in the Playlist and set the Loupedeck to send MIDI on the Performance channel (usually MIDI channel 1) ([Playlist in Performance Mode](https://www.image-line.com/fl-studio-learning/fl-studio-online-manual/html/playlist_performance.htm#:~:text=Performance%20Mode%2C%20see%20Mapping%20non,controllers)) ([Playlist in Performance Mode](https://www.image-line.com/fl-studio-learning/fl-studio-online-manual/html/playlist_performance.htm#:~:text=MIDI%20notes%20pre,bar%20starting%20with%20each%20marker)). Where the provided icon packs didn’t have an obvious symbol (e.g., FL’s unique tools like **Ghost notes** or **Pattern/Song toggle**), we recommended custom icons – you can create simple PNGs or SVGs (even using text or FL Studio’s icons) to import into Loupedeck. This comprehensive setup should cover most of your FL Studio workflow: from composing in the Piano Roll and Channel Rack, arranging in the Playlist, mixing, editing audio in Edison, to performing with live pads.
